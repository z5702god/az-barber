## 提交流程

### Step 1: 建置正式版本
```bash
eas build --platform ios --profile production
```

### Step 2: 提交到 App Store Connect
```bash
eas submit --platform ios
```

### Step 3: 在 App Store Connect 填寫資料
1. 登入 https://appstoreconnect.apple.com
2. 選擇「AZ Barber」App
3. 填入以上所有中繼資料
4. 上傳截圖
5. 填寫年齡分級問卷
6. 填寫審核備註與測試帳號
7. 點擊「提交審核」

### Step 4: 等待審核
- 通常需要 24-48 小時
- 首次提交可能需要較長時間
- 留意 App Store Connect 的通知信
